## ✨ 웹페이지 주요 특징

* **반응형 웹 디자인 (Responsive Web Design)**: 모바일, 태블릿, 데스크톱 등 다양한 기기 화면 크기에 맞추어 레이아웃이 최적화됩니다.
* **다크 모드 대응 (Dark Mode Support)**: 시스템 설정(다크 모드/라이트 모드)에 반응하여 자동으로 테마가 전환됩니다.
* **직관적인 UI/UX**: 가독성 높은 폰트 조합과 카드 레이아웃을 통해 정보를 명확하게 전달합니다.

## 🛠️ 개발 및 호스팅 안내

본 저장소의 HTML 파일들은 서버 사이드 로직이 없는 순수 정적(Static) 웹페이지입니다.  
GitHub Pages, Vercel, Netlify 또는 AWS S3 등의 정적 호스팅 서비스를 통해 즉시 배포하여 사용할 수 있습니다.
